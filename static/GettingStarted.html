Intro 🎉

A modern style notebook based on Tiddlywiki. @oeyoews

Features ✨

  • 📰 Support Atome Feed(Rss)
  • 👤 Support gravatar
  • 🎨 Support use tailwindcss to better typography
  • 🔀 Randomly display articles
  • 💻 Support F fullscreen, s toggle fullscreen
  • 🌐 Article titles support bilingual language switching
  • ✏️ Support switching between reading mode and editing mode
  • 😄 Support emoji syntax
  • 📐 Support switching between multiple layouts via URL
  • 🌙 Automatically switch between night/day mode based on system settings, support URL switching
  • 🧘 Support Zen mode
  • 💬 Support comments
  • 📆 Automatically display recently created articles
  • 📝 Automatically add spaces between Chinese and English for better typography
  • 🏷️ Support title aliases

Installation 📦

git clone --depth 1 https://github.com/oeyoews/neotw
cd neotw && yarn or yarn install  # install packages
yarn start  # start tiddlywiki on https://localhost:8099 or use pm2 with yarn pm2:start
