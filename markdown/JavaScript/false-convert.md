# false-convert

转换规则是除了下面六个值被转为 false，其他值都视为 true。

* undefined

* null

* false

* 0

* NaN

* ""或’'（空字符串）

:::note\

注意，空数组（[]）和空对象（{}）对应的布尔值，都是 true。\

:::
