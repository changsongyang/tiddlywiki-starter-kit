# js-data-type

> <https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Language_overview>

## Data type

* Number（数字）
* String（字符串）
* Boolean（布尔）
* Symbol（符号）（ES2015 新增）
* Object（对象）
    * Function（函数）
    * Array（数组）
    * Date（日期）
    * RegExp（正则表达式）
* null（空）
* undefined（未定义）
